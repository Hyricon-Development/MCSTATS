# MCSTATS









**INSTALLATION**
`git clone https://github.com/Evolution-Development/MCSTATS.git`
<br>
`cd MCSTATS`
<br>
`npm install`
EDIT `config.yml`
`mv .env.example .env`
EDIT `.env`

<hr>

**TO START THE BOT**
1. First test for any errors using `node index.js`
2. Then start MCSTATS using `pm2 start index.js`
